## 1. NL2SQL Hardening

- **Keyword Blacklisting**: Blocks `DROP`, `DELETE`, `TRUNCATE`, `ALTER`, `GRANT`, `REVOKE`.
- **Injection Pattern Detection**: Regex-based detection of semicolon-linked statements (e.g., `SELECT ...; DROP ...`).
- **Sink Sanitization**: Final query execution is wrapped in a validation gate in `ai_handler.go`.
